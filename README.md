    Настя, студентка, с тетрадкой и ручкой,

    В мир JavaScript шагнула, не зная, что ждет.

    Сначала казалось: "Ну что это за штучка?

    Зачем эти скобки, где точка, где код?"


    const, let, var — всё смешалось вначале,

    forEach, map, filter — как будто язык.

    Но Настя не сдавалась, хоть нервы кричали,

    И каждый console.log как маленький блик.


    И вот уже DOM ей послушен, как глина,

    sessionStorage хранит её мемы-шедевры.

    Теперь JavaScript — не чужая чужбина,

    А мощный инструмент, что бережет её нервы!


    C улыбкой Настюша глядит на свой код,

    Ведь каждый return true — это маленький взлет.

    И знает теперь: JavaScript — это сила,

    Когда ты упорно идешь напролом!
